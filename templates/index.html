doors.modeemi.fi is a service meant mainly for a student associations at Tampere University, which shows is the association space is open, so You know if it's possible to visit the space. The web application was created by Tsuribori, an IoT part by Niko, idea proposed by Rowan and the server is hosted on Modeemi Ry's infrastructure, an IT Hobby Club operating at Tampere University (form. Tampere University of Technology) since 1977.
