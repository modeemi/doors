About

doors.modeemi.fi is a service meant mainly for a student associations at Tampere University, which shows if the association space is open, so You know if it's possible to visit the space. This is basically next iteration of project "Hervanta-Bots". If You wanna know more about the projects, read our technical description.

Credits
  • Tuomas "Tsuribori" - main web developer
  • Nikoloz "Niko" Glonti - IoT developer + assistant web developer + Sysadmin
  • Rowan van der Kaaden - developer of the original "Hervanta-bots"
Project is part of Modeemi Ry's services. Modeemi is an IT Hobby Club operating at Tampere University (form. Tampere University of Technology) since 1977.
